# Bike registration example (over lwt)

This simple program demonstrates:

- creating a temporary table
- inserting rows
- selecting rows
- deleting rows

```sh
$ PGHOST=localhost PGDATABASE=caqti_study PGPORT=5433 ./bikereg.exe
```
